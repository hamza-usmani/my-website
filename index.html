Quickplan

A M*EAN (*MySQL, Express, Angular, Node)Â application developed as an on-the-fly event planning platform. A fully responsive application which features JWT (JSON Web Token) authentication for quick and secure sign-up/login, incorporates Google Maps API, and presents a specifically designed event-calendar. This app allows users to create and share plans, and already hosts over 100 users!

Front-end uses Angular5/Typescript 2.5.3 with Bootstrap 4 and Material design. Third-party tools such as Froala Editor, Clipboard Module and Google Maps incorporated into the planning app.

NodeJS (Express Framework) backend which handles authentication and user-data requests. Utilizes Passport for Node (authentication middleware), JWT, and UUID-generation for sharable links, and CORS (Cross Origin Request) to allow HttpClient requests. MySQL database used as the data layer, with sha512-encryption, and Sequelize used as a promise-based ORM.
