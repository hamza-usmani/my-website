Quadris

A game similar to Tetris (with modified/new rules) created in C++, using graphics in XWindow for UI. Program utilizes MVC Design, Factory Method Pattern, Observer Pattern, several standard library methods, and has inheritance play a major role in block creation.

The design consists of a game board partitioned by cells, in which each cell has a state. This board can contain different types of blocks, which can be any block derived from the virtual block class. Blocks can be rotated both clockwise and counter-clockwise, moved left, right, down and dropped upon userâs request. The Level classes specify different difficulties in which the game can be played. Probabilities involved in block generation are dependent on level as well as the ability to take in a seed for random generation.

The Levels and the Board classes employ the factory method pattern and the Cells and both Text Display and Graphical Display interact following the observer pattern. The cells behave as the subject meanwhile the other two classes observe and wait to be notified.
